I used CSS cosmetic filters which can be used in uBlock origin and Lite custom filters. This collection of uBlock Origin filters creates a refined, experience for the Gemini interface into a cleaner, dark-themed contrast elements with the following core improvements:

* Layout & Width: Widens the conversation and input columns using responsive constraints for better readability on large screens.
* Visual Cleanup: Removes distracting UI elements, including gradients behind the input box, hallucination disclaimers, and unnecessary sidebar buttons.
* Enhanced Styling: Applies a consistent dark aesthetic, featuring forest green user chat bubbles, pure black backgrounds for code blocks, and high-contrast borders for image previews.
* Table & Scrollbar Fixes: Eliminates table-fading gradients and glitches, while ensuring full table content visibility and persistent scrollbars.
